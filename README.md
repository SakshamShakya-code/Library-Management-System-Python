# Library Management System

A GUI-based Library Management System developed using Python, Tkinter, and MySQL.

## Features
- Add Book
- Delete Book
- View Books
- Issue Book
- Return Book

## Technologies Used
- Python
- Tkinter
- MySQL
- PyMySQL
- Pillow

## Project Structure
- Modular Python files for each functionality
- Database integration for storing and managing records

## How to Run
1. Install required libraries:
```bash
pip install -r requirements.txt
```

2. Configure MySQL database credentials in the Python files

3. Run:
```bash
python main.py
```
